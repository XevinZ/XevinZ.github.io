<h1>Steps on how to create a github page</h1>
<p>1. Create a new public repository with a README file</p>
<p>2. Add your files on the repository. The html file should be named index.html</p>
<p>3. Go to you repository and click Settings.After that, go to Pages and under source you should deploy from a branch</p>
<p>4. Visit your website by going to yourusername.github.io </p>
